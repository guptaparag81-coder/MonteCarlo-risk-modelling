# Monte Carlo Risk Modelling

A quantitative finance project demonstrating Monte Carlo simulation and Value-at-Risk (VaR) estimation for portfolio risk analysis.

## Objective

The purpose of this project is to estimate financial risk using Monte Carlo simulation techniques.

The project generates thousands of possible future portfolio outcomes using stochastic modelling and estimates Value-at-Risk (VaR), a widely used financial risk metric.

## Features

- Monte Carlo simulation
- Stochastic modelling
- Portfolio risk analysis
- Value-at-Risk (VaR) estimation
- Probability-based forecasting
- Data visualisation

## Technologies

- Python
- NumPy
- Pandas
- Matplotlib

## Applications

- Financial Risk Management
- Quantitative Finance
- Portfolio Analysis
- Investment Banking
- Asset Management
- Risk Modelling
